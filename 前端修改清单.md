# 前端代码修改清单

## ⚠️ 重要：构建前端前必须修改

在部署到生产环境前，需要修改以下前端代码：

---

## 1. 企业微信登录回调地址

**文件**: `frontend/src/config/index.js`

**当前代码**:
```javascript
export const config = {
  redirectUri: "https://abc123xyz.ngrok.io/login",
};
```

**修改为**:
```javascript
export const config = {
  redirectUri: "https://bjcastapp.bijienetworks.com/login",
};
```

**说明**: 这是企业微信OAuth登录成功后的回调地址，必须与企业微信后台配置的可信域名一致。

---

## 2. 音频文件访问地址

**文件**: `frontend/src/views/meetings/detail.vue`

**当前代码** (第205行):
```javascript
return "http://192.168.11.16:8080" + audioUrl;
```

**修改为**:
```javascript
return "https://bjcastapp.bijienetworks.com" + audioUrl;
```

**说明**: 这是前端访问后端音频文件时的地址，需要改成 HTTPS。

---

## 修改步骤

### 方式一：直接修改后重新构建

```bash
# 1. 修改前端代码
# 编辑 frontend/src/config/index.js
# 编辑 frontend/src/views/meetings/detail.vue

# 2. 重新构建前端
cd frontend
npm run build

# 3. 上传到服务器
scp -r dist/* root@你的服务器IP:/opt/wechat_meeting/dist/
```

### 方式二：直接修改 dist 目录（已构建的版本）

如果前端已经构建完成，可以直接修改 `dist/index.html` 中的配置：

```bash
# 1. 查找并替换 dist/index.html 中的地址
sed -i 's/http:\/\/192\.168\.11\.16:8080/https:\/\/bjcastapp.bijienetworks.com/g' dist/index.html

# 2. 上传到服务器
scp -r dist/* root@你的服务器IP:/opt/wechat_meeting/dist/
```

---

## 验证修改

修改完成后，检查以下内容：

1. **企业微信回调地址**
   - 企业微信后台配置的可信域名: `bjcastapp.bijienetworks.com`
   - 前端 redirectUri: `https://bjcastapp.bijienetworks.com/login`

2. **API 调用**
   - 前端 API baseURL: `/api`（相对路径，通过 Nginx 代理）
   - Nginx 会将 `/api` 转发到 `http://127.0.0.1:8090`

3. **音频文件访问**
   - 前端访问音频: `https://bjcastapp.bijienetworks.com/uploads/xxx.mp3`
   - Nginx 会将 `/uploads/` 转发到 `/opt/wechat_meeting/uploads/`

---

## Nginx 配置对应的路由规则

```nginx
# API 请求 -> 后端 8090 端口
location /api/ {
    proxy_pass http://127.0.0.1:8090;
    ...
}

# 音频文件 -> 上传目录
location /uploads/ {
    alias /opt/wechat_meeting/uploads/;
    ...
}

# 其他请求 -> 前端 dist 目录
location / {
    root /opt/wechat_meeting/dist;
    index index.html;
    try_files $uri $uri/ /index.html;
}
```

---

## ⚠️ 注意事项

1. **必须使用 HTTPS**: 企业微信要求所有回调地址必须使用 HTTPS
2. **域名一致性**: 企业微信后台配置的可信域名必须与 redirectUri 匹配
3. **重新构建**: 修改前端代码后必须重新 `npm run build`
4. **上传 dist 目录**: 确保上传的是最新的构建结果

---

## 快速命令

### Linux 服务器上直接修改（推荐方式一）

```bash
# 在服务器上修改
sed -i 's|http://192\.168\.11\.16:8080|https://bjcastapp.bijienetworks.com|g' /opt/wechat_meeting/dist/index.html
sed -i 's|https://abc123xyz\.ngrok\.io|https://bjcastapp.bijienetworks.com|g' /opt/wechat_meeting/dist/index.html

# 重启 Nginx
systemctl restart nginx
```

### Windows 本地修改后重新上传（推荐方式二）

```bash
# 1. 修改 frontend/src/config/index.js
# 2. 修改 frontend/src/views/meetings/detail.vue
# 3. 重新构建
cd frontend
npm run build

# 4. 上传
scp -r dist/* root@你的服务器IP:/opt/wechat_meeting/dist/
```
