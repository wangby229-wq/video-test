## 第三部分：创建 Supervisor 配置

### 先查看现有配置作为参考：
```bash
ssh ecs-user@47.96.21.34
cat /etc/supervisord.d/java8889.ini
exit
```

### 创建新的 Supervisor 配置（端口 8090）：
```bash
ssh ecs-user@47.96.21.34
sudo su -
cat > /etc/supervisord.d/java8090.ini << 'EOF'
[program:meeting-server]
directory=/opt/wechat_meeting
command=/usr/bin/java -Xms256m -Xmx512m -jar meeting-transcribe-1.0.0.jar --spring.config.location=/opt/wechat_meeting/application.yml
numprocs=1
autostart=true
autorestart=true
startsecs=10
startretries=3
exitcodes=0
stopsignal=TERM
stopwaitsecs=60
user=root
redirect_stderr=true
stdout_logfile=/var/log/wechat_meeting/meeting-server.log
stdout_logfile_maxbytes=50MB
stdout_logfile_backups=10
EOF
exit
```

### 启动服务：
```bash
ssh ecs-user@47.96.21.34
supervisorctl reread
supervisorctl update
supervisorctl start meeting-server
supervisorctl status meeting-server
```

### 查看日志：
```bash
ssh ecs-user@47.96.21.34
tail -f /var/log/wechat_meeting/meeting-server.log
```

## 第四部分：配置 Nginx

### 查看现有配置：
```bash
ssh ecs-user@47.96.21.34
cat /etc/nginx/nginx.conf | head -50
ls -la /etc/nginx/conf.d/
cat /etc/nginx/conf.d/bjcastapp.conf
exit
```

### 添加 Nginx 配置：
```bash
ssh ecs-user@47.96.21.34
sudo su -
cat > /etc/nginx/conf.d/meeting.conf << 'EOF'
# 会议转录系统前端
location / {
    root /opt/wechat_meeting/dist;
    index index.html;
    try_files $uri $uri/ /index.html;
}

# 会议转录系统后端 API
location /api/ {
    proxy_pass http://127.0.0.1:8090;
    proxy_http_version 1.1;
    proxy_set_header Host $host;
    proxy_set_header X-Real-IP $remote_addr;
    proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    proxy_set_header X-Forwarded-Proto $scheme;
    client_max_body_size 500M;
    proxy_read_timeout 300s;
    proxy_connect_timeout 75s;
}
EOF
exit
```

### 重启 Nginx：
```bash
ssh ecs-user@47.96.21.34
sudo nginx -t
sudo systemctl restart nginx
sudo systemctl status nginx
```
